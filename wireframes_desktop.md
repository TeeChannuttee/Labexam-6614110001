```

│  ┌──────────┬────────────────────────────────────────────┐  │

│  │  SIDE    │           MAIN CONTENT                     │  │

│  │  [👔]    │  Add Look:     2-column form + preview    │  │
│  │  [➕]    │  Swipe Mode:   Center card + side preview │  │
```

## Desktop-Specific Features

| Feature | Implementation |
|---------|----------------|
| **Sidebar Navigation** | Fixed left sidebar, always visible |
| **Grid Layout** | 5 columns instead of mobile's 1 column |
| **Keyboard Shortcuts** | Arrow keys for swipe, Enter to add |
| **Drag & Drop** | For reordering and file upload |
| **Hover Effects** | Visual feedback on all interactive elements |
| **Wide Screen** | Optimized for 1200px - 1920px width |
| **Multi-column Forms** | Side-by-side layout for efficiency |
| **Persistent Filters** | Always visible, no collapse needed |
