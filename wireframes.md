# Outfit Tinder - Wireframes

## ภาพรวมการนำทาง (Navigation Flow)

```
┌─────────────┐         ┌─────────────┐         ┌─────────────┐
│ All Looks   │  ──+──> │  Add Look   │         │ Swipe Mode  │

                        Back / Navigation Bar
```

## 1️⃣ All Looks (คลังชุด)

**ฟังก์ชันหลัก:**
- แสดงรายการลุคทั้งหมด
- ปัดซ้ายเพื่อลบ (Dismissible)
- ลากจัดเรียงใหม่ (Reorderable)

```
┌────────────────────────────────────┐
│ ← All Looks              [+]       │  ← Header + ปุ่มเพิ่มลุค
├────────────────────────────────────┤
│                                    │
│  ┌──────────────────────────────┐ │
│  │ ┌────┐  Casual Weekend    ≡≡ │ │  ← รายการลุค + Drag Handle
│  │ │IMG │  สไตล์: Minimal        │ │
│  │ │    │  ความมั่นใจ: ★★★★☆    │ │
│  │ └────┘                        │ │
│  │ ←swipe to delete              │ │  ← แสดงว่าปัดซ้ายได้
│  └──────────────────────────────┘ │
│                                    │
│  ┌──────────────────────────────┐ │
│  │ ┌────┐  Street Style      ≡≡ │ │
│  │ │IMG │  สไตล์: Street         │ │
│  │ │    │  ความมั่นใจ: ★★★☆☆    │ │
│  │ └────┘                        │ │
│  └──────────────────────────────┘ │
│                                    │
│  ┌──────────────────────────────┐ │
│  │ ┌────┐  Korean Chic       ≡≡ │ │
│  │ │IMG │  สไตล์: Korean         │ │
│  │ │    │  ความมั่นใจ: ★★★★★    │ │
│  │ └────┘                        │ │
│  └──────────────────────────────┘ │
│                                    │
│         ▼ เลื่อนลงดูเพิ่ม ▼       │
│                                    │
├────────────────────────────────────┤
│  [👔 All]  [➕ Add]  [👆 Swipe]  │  ← Bottom Navigation
└────────────────────────────────────┘
```

**องค์ประกอบ:**
- ✅ Header พร้อมชื่อหน้าจอและปุ่ม [+]
- ✅ รายการ Cards แสดงรูป + ชื่อ + สไตล์ + ความมั่นใจ
- ✅ Icon ≡≡ สำหรับลากจัดเรียง
- ✅ รองรับ Swipe-to-Delete
- ✅ Bottom Navigation Bar (3 tabs)

## 2️⃣ Add Look (เพิ่มลุค)

**ฟังก์ชันหลัก:**
- ฟอร์มเพิ่มลุคใหม่
- Validation สำหรับทุกฟิลด์
- เลือก/ถ่ายรูปภาพ

```
┌────────────────────────────────────┐
│ ← Add Look                         │  ← Header + Back Arrow
├────────────────────────────────────┤
│                                    │
│  ชื่อชุด *                         │
│  ┌──────────────────────────────┐ │
│  │ Casual Weekend               │ │  ← Text Input
│  └──────────────────────────────┘ │
│  (min 2 ตัวอักษร)                 │
│                                    │
│  สไตล์ *                           │
│  ┌──────────────────────────────┐ │
│  │ Minimal              ▼       │ │  ← Dropdown
│  └──────────────────────────────┘ │
│  ⌞ minimal / street / korean      │
│    / vintage                       │
│                                    │
│  ระดับความมั่นใจ * (1-5)          │
│  ┌──────────────────────────────┐ │
│  │ 4                            │ │  ← Number Input
│  └──────────────────────────────┘ │
│  ⭐⭐⭐⭐☆                         │
│                                    │
│  รูปภาพลุค *                      │
│  ┌──────────────────────────────┐ │
│  │                              │ │
│  │         📷                   │ │  ← Image Picker
│  │   เลือกรูป / ถ่ายรูป         │ │
│  │                              │ │
│  └──────────────────────────────┘ │
│                                    │
│  ┌──────────────────────────────┐ │
│  │        💾 บันทึก             │ │  ← Save Button
│  └──────────────────────────────┘ │
│                                    │
├────────────────────────────────────┤
│  [👔 All]  [➕ Add]  [👆 Swipe]  │  ← Bottom Navigation
└────────────────────────────────────┘
```

**องค์ประกอบ:**
- ✅ Header พร้อม Back Button
- ✅ 4 Input Fields พร้อม Validation:
  1. **ชื่อชุด** (TextField, required, min 2 chars)
  2. **สไตล์** (Dropdown, required)
  3. **ระดับความมั่นใจ** (Number 1-5, required)
  4. **รูปภาพ** (Image Picker, required)
- ✅ ปุ่มบันทึก
- ✅ Bottom Navigation Bar

## 3️⃣ Swipe Mode (โหมดปัด)

**ฟังก์ชันหลัก:**
- แสดงลุคทีละชุด
- ปัดซ้าย = ไม่ใช่
- ปัดขวา = ใช่

```
┌────────────────────────────────────┐
│ ← Swipe Mode                       │  ← Header + Back Arrow
├────────────────────────────────────┤
│                                    │
│                                    │
│    ┌──────────────────────────┐   │
│    │                          │   │  
│    │      [OUTFIT IMAGE]      │   │  ← Large Card
│    │                          │   │     แสดงรูปลุค
│    │                          │   │
│    │      ┌────────────────┐  │   │
│    │      │ Casual Weekend │  │   │
│    │      │ สไตล์: Minimal │  │   │  ← ข้อมูลลุค
│    │      └────────────────┘  │   │
│    └──────────────────────────┘   │
│   ╱                            ╲  │
│  ❌                              ✅ │  ← Swipe Indicators
│ ไม่ใช่                         ใช่ │
│                                    │
│        👈 swipe 👉                │  ← คำแนะนำ
│                                    │
│    [card] [card]  ← stack preview  │  ← การ์ดถัดไปซ้อนอยู่ด้านหลัง
│                                    │
├────────────────────────────────────┤
│  [👔 All]  [➕ Add]  [👆 Swipe]  │  ← Bottom Navigation
└────────────────────────────────────┘
```

**องค์ประกอบ:**
- ✅ Header พร้อม Back Button
- ✅ Large Card แสดงรูปภาพลุคขนาดใหญ่
- ✅ ข้อมูลลุค (ชื่อ + สไตล์) แสดงด้านล่างรูป
- ✅ Swipe Gesture Support:
  - **ปัดซ้าย** → ❌ ไม่ใช่
  - **ปัดขวา** → ✅ ใช่
- ✅ Stack ของการ์ดที่เหลือแสดงเบาๆ ด้านหลัง
- ✅ Bottom Navigation Bar

## สรุปการนำทาง (Navigation Summary)

| จาก Screen | ไป Screen | Action |
|------------|-----------|--------|
| **All Looks** | Add Look | คลิกปุ่ม [+] ที่ Header |
| **All Looks** | Swipe Mode | คลิก Tab [Swipe] ที่ Bottom Nav |
| **Add Look** | All Looks | กดปุ่ม [บันทึก] หรือ [←] |
| **Swipe Mode** | All Looks | กดปุ่ม [←] หรือ Tab [All] |
| **ทุกหน้า** | ทุกหน้า | ใช้ Bottom Navigation Bar |

## Key Features Mapping

| Feature | Screen | Component |
|---------|--------|-----------|
| **Dismissible** | All Looks | Swipe left to delete card |
| **Reorderable** | All Looks | Drag ≡≡ icon to reorder |
| **Validation** | Add Look | 3 required fields + image |
| **Swipe Gesture** | Swipe Mode | Swipeable card (left/right) |
| **Image Picker** | Add Look | Camera/Gallery selection |
| **Value Key** | All Looks | `ValueKey(look.id)` for list items |
